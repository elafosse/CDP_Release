Create an Test object.(id, projectId, name, description, resultExpected, state, listIssues)

new Create an Test object.(id, projectId, name, description, resultExpected, state, listIssues)

Represent an test.
Parameters:
Name Type Description
id int The id of the Test object. It is unique and is generated by the database.
projectId int The id of the project the test belongs to.
name string The name of the test.
description string The description of the test.
resultExpected string The results we should have if the test is passed.
state string The current state of the test. It should be "todo", "passed" or "failed". By default, it should be "todo".
listIssues Array The list of the issues taht should be implemented at the end of the sprint.
