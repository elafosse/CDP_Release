Create a Task object.(id, projectId, name, description, state, start_date, realisation_time, dod, dependancies, members, issues)

new Create a Task object.(id, projectId, name, description, state, start_date, realisation_time, dod, dependancies, members, issues)

Represent a task.
Parameters:
Name Type Description
id int The id of the task. It is generated by the database.
projectId int The id of the project the task belongs to.
name string The name of the task.
description string The description of the task.
state string The current state of the task. It should be "to do", "doing" or "done".
start_date date The date the implementation of the task should start.
realisation_time int The time in "jh" estimated to implement this task. It should be an integer greater or equal to 0.
dod string The Definition of Done of the task.
dependancies Array The list of tasks that need to be done before starting this one.
members Array The members of the project assigned to this task.
issues Array The list of issues this task implements.
