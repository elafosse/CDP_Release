Create a Member object.(username, password, listProjects)

new Create a Member object.(username, password, listProjects)

Represent the account of a user.
Parameters:
Name Type Description
username string The username of the user. It is unique.
password string The password of the user. It can be the same for different members.
listProjects Array The list of the projects the user is taking part in. It does not matter if the user is an administrator of the project or not.
